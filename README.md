# AI-Data-Analysis-Assistant

An interactive web application built with Python, Pandas, and Streamlit that allows users to upload datasets and automatically perform data analysis, cleaning, and visualization with AI-generated insights. It is designed to make data analysis fast, simple and accessible.

## Contents
- **app.py** → This is the main file that runs the Streamlit app
- **cleaning.py** → This file is responsible for preprocessing the raw dataset makes it clean and usable for analysis
- **visualization.py** → This file creates all graphs and charts
-  **insights.py** → This file generates automated explanations from the dataset

## Tools Used
- python
- pandas
- streamlit
- matplotlib
- numpy
